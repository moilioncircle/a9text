3.2. impl_javascript a9engine example
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
In fact, the a9engine is a class loader of a9text,it make js works like java.
it is an important part of a9text runtime,it can works standalong.

code example:

require('com.trydofor.a9text.render.dom.a9text');
require('com.trydofor.a9text.render.html.a9text');
var __Renders__ = {};
__Renders__.dom = new com.trydofor.a9text.render.dom.a9text.A9TextRender();
__Renders__.htm = new com.trydofor.a9text.render.html.a9text.A9TextRender();
